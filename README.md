# 🤖 AI SaaS Platform

A full-stack **AI-powered SaaS web application** that provides multiple AI tools such as Article generation, image generate, background remove, object remove, resume checking features with secure authentication and subscription-based access.

---

## Live Demo

🔗 https://ai-saas-frontend-nk3k.onrender.com/

---

## 🛠 Tech Stack

**Frontend**

* React.js
* Tailwind CSS
* Prebuild ui

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB & NeonDB

**Payments**

* Razorpay

**Authentication**

* Clerk Authentication

**AI Integration**

* Gimini

---

## ✨ Features

### 👤 User Features

* 🔐 Secure authentication (Clerk)
* 🧠 Access multiple AI tools
* ✍️ AI text/content generation
* 📄 Save generated results
* 📊 Usage-based access system

---

### 💼 SaaS Features

* 💳 Subscription-based model (Razorpay integration)
* 📈 User usage tracking
* 🔒 Protected routes (premium vs free users)
* 🧾 Billing & plan management

---

### ⚡ AI Functionalities

* 📝 Image generation
* 💡 Article writing
* 📷 Image Generate
* ✏️ Object Remove from image
* 📊 Resume checking

---

### 🧑‍💼 Admin / System Features

* 👥 User management
* 📊 Monitor usage & subscriptions
* ⚙️ API handling & rate limiting
